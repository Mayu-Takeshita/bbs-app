# Flask 掲示板システム (BBS)

PythonのFlaskフレームワークとSQLiteを使用した、階層構造型掲示板システムです。
「板（Board） > スレッド（Thread） > 投稿（Post）」のデータ構造を持ち、ユーザー認証（登録・ログイン）機能を備えています。

##　🚀　主な機能

- **ユーザー認証**: パスワードハッシュ化（PBKDF2）によるセキュアなユーザー登録・ログイン
- **階層構造管理**: 複数の掲示板（板）の切り替え、スレッド一覧の表示
- **スレッド・投稿機能**: 任意の板へのスレッド作成、投稿およびタイムスタンプの自動付与
- **返信機能**: 特定の投稿IDを指定したアンカー付きの返信投稿
- **データ永続化**: SQLiteを使用したデータの保存。本リポジトリには初期データ入りの `bbs.db` を同梱しています。

## 🛠 技術スタック

- **言語**: Python 3.9+
- **Webフレームワーク**: Flask
- **データベース**: SQLite / SQLAlchemy (ORM)
- **認証**: Flask-Login
- **UI**: Bootstrap 5

## ⚙️ セットアップと実行方法
1. ライブラリのインストール
ターミナルで以下のコマンドを実行して、必要なパッケージをインストールします。

```Bash`
pip3 install -r requirements.txt
2. アプリケーションの実行
```Bash`
python3 app.py
実行後、ブラウザで http://127.0.0.1:5000 にアクセスしてください。

3. 初期データ
bbs.db にはあらかじめ「ニュース」「テクノロジー」「雑談」の3つの板が用意されています。

## 📝 開発上の注意点
セキュリティ: app.py 内の SECRET_KEY は開発用の値です。本番環境で公開する場合は環境変数に切り替えてください。

データベース: bbs.db をリセットしたい場合は、ファイルを削除してアプリを再起動すれば、空のテーブルが自動的に再生成されます。
